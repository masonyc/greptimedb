pub fn trace_id() -> Option<u64>
Expand description

Get current TRACE_ID from tokio task_local storage.

Usage

To set current trace id, wrap your async code like this:

common_telemetry::TRACE_ID
    .scope(id, async move {
        query_handler
            .do_query(query, self.session.context())
            .await
    })
    .await

Then all functions called from this stack will be able to retrieve the trace id via this method.
